# betcom
mobile application for android and ios building by react native , user can rent and booking apartment online and else can  offered his units for rent
## step 1
to clone this repo on your local device
run `git clone https://github.com/Hazem-Alsaqaan/betcom.git`
## step 2
to install dependences
run `npm install`
## step 3
to display the app
run `npx expo start`
